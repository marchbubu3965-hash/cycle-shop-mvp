# ♻️ 兒童衣物循環商城 (Green Cycle MVP)

這是一個基於 **Node.js** 和 **SQLite** 開發的輕量級電子商務系統 MVP（最小可行性產品）。本專案旨在解決兒童衣物汰換過快的問題，透過「低庫存預購」與「官方回購機制」，達成永續循環的商業模式。

## 🌟 核心功能

### 👤 使用者端 (Buyer)
- **自主註冊與安全登入**：支援買家建立帳號，並具備密碼驗證機制。
- **商品瀏覽與搜尋**：即時查看上架商品，支援名稱關鍵字篩選。
- **購物車系統**：流暢的加入購物車與結帳流程。
- **混合支付機制**：系統自動優先扣除「購物金（Wallet）」，不足額部分顯示現金實付金額。
- **訂單紀錄與回購申請**：完整記錄購買歷史，並可針對持有中的商品向管理員發起「回購申請」。
- **回購金查看**：紀錄已完成回購的項目與獲取的獎勵金額。

### 🛠 管理員端 (Admin)
- **庫存管理**：即時更新商品庫存數量。
- **商品上架**：支援圖片上傳（Multer）與價格、庫存設定。
- **回購審核系統**：查看買家發起的回購申請，手動輸入「核發金額」並撥款至買家錢包。

## 🚀 技術棧

- **前端**: 原生 HTML5, CSS3 (Modern Flex/Grid Layout), Vanilla JavaScript (Async/Await)
- **後端**: Node.js, Express.js
- **資料庫**: SQLite3 (關聯式資料庫設計)
- **檔案處理**: Multer (圖片上傳處理)
- **版本控制**: Git
